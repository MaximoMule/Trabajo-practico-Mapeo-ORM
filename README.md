# Trabajo-practico-Mapeo-ORM


## Descripción
Trabajo práctico de la materia donde se aplican conceptos básicos de **Mapeo Objeto–Relacional (ORM)** utilizando **Java** y **JPA**.

El proyecto modela un sistema simple de hospital con pacientes, personal, departamentos y hospitales, utilizando distintas anotaciones de persistencia.

## Tecnologías
- Java  
- Spring Boot  
- JPA (Jakarta Persistence)  
- Hibernate  

## Contenido
- Clases embebibles (`Address`, `Phone`)
- Herencia con `@MappedSuperclass`
- Entidades (`Hospital`, `Department`, `Patient`, `Staff`)
- Relaciones entre entidades
- Uso de `@ElementCollection`

## Ejecución
1. Importar el proyecto en el IDE
2. Ejecutar la aplicación desde `HospitalApplication`
